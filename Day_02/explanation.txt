My blood_moon function calculates the next three Blood Moon times by converting the input time into total minutes, then repeatedly adding the 168‑minute interval at which Blood Moons occur. 

After splitting the given "HH:MM" time into hours and minutes, it converts that into total minutes since midnight so calculations are easy. Each new Blood Moon time is found by adding 168 minutes and wrapping around the day using modulo 1440 (the number of minutes in a day). The result is then converted back into a "HH:MM" format with leading zeros and added to a list. 

By updating the start time after each calculation, the function generates a sequence of three upcoming Blood Moons based on the initial time.
