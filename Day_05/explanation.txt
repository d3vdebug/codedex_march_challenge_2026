My calculate_score function computes the final figure‑skating element score by taking each element’s base value and adjusting it using a refined Grade of Execution (GOE). 

For every element, it starts by sorting the list of nine GOE scores, removing the highest and lowest to eliminate bias, and averaging the remaining seven values. 

This average GOE is then scaled using the formula avg_goe * 0.1 * base, which reflects how GOE modifies the base value in real scoring systems. The function adds this adjusted score to the element’s base to get the element’s total value, updates the running total, and finally returns the overall score rounded to one decimal place.
