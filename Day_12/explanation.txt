My check_palindrome function checks whether a given string is a palindrome by first normalizing the input. It removes all spaces using replace(" ", "") and converts everything to lowercase so that capitalization doesn’t affect the result. This gives us a clean version of the text that can be compared fairly. To determine whether it’s a palindrome, the function simply compares the cleaned string to its reverse, which in Python is written as text[::-1]. If both sides match, the function returns True; otherwise, it returns False.
