In this solution, I simulate how the green dye spreads through the river over time. First, I create a copy of the original river list so the initial state remains unchanged. 

Then I loop through every position in the river to check where the dye (☘️) starts. Whenever a dye patch is found, another loop spreads the dye to the right for the given number of hours. 

Each step calculates the new position (i + j) where the dye should drift, and if that position is still inside the river, it turns that spot green in the result list. 

Finally, the function returns the updated river showing how the dye has spread downstream after the specified number of hours.
