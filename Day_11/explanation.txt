My function checks whether a string is a valid web address based on a few simple rules. First, it ensures the URL starts with either http:// or https://. If it doesn’t, the function immediately returns "invalid". Next, it extracts everything after the scheme by splitting on "://", which gives us the rest of the address. If the domain begins with www., that prefix is removed to make the domain easier to validate. Then it performs two quick checks on the domain: it must not contain any spaces, and it must contain at least one dot (.) (e.g., example.com). If either check fails, it returns "invalid". Otherwise, it returns "valid".
